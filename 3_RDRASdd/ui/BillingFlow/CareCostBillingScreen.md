# UI仕様書: 介護費用請求画面

**画面日本語名**: 介護費用請求画面  
**画面英語名**: CareCostBillingScreen  
**所属BUC**: BillingFlow  

**目的**: 計算された介護費用を確認し、請求内容を検証してから請求を実行。請求状況を管理。

---

## 表示/入力項目

| 項目名 | 型 | 必須 | 編集可否 |
|-------|---|-----|--------|
| 請求ID | number | - | - |
| 会員名 | string | - | - |
| 請求月 | string | - | - |
| 請求金額 | number | - | - |
| 支払期限 | date | ○ | ○ |

---

## API利用

**請求**: POST /api/v1/invoices/execute

**リクエスト**:
```json
{
  "costId": "number",
  "dueDate": "string (YYYY-MM-DD)"
}
```

---

## 操作フロー

```
START → [請求対象費用一覧表示]
  → [費用選択] → 詳細表示
  → [支払期限を設定]
  → [請求ボタン押下] → POST /execute
    ├→ [成功] → 成功メッセージ
    └→ [失敗] → エラーメッセージ
  → [請求状況を管理]
  → END
```

---

## 機能要求

| 要求ID | 要求内容 | 優先度 |
|-------|--------|-------|
| FR-001 | 費用を確認できること | 必須 |
