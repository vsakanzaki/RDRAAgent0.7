# UI仕様書: 介護費用計算画面

**画面日本語名**: 介護費用計算画面  
**画面英語名**: CareCostCalculationScreen  
**所属BUC**: CareCostCalculationFlow  

**目的**: 実施記録に基づいて月別の介護費用を計算。計算結果を確認・修正・承認。

---

## 表示/入力項目

| 項目名 | 型 | 必須 | 編集可否 |
|-------|---|-----|--------|
| 計算ID | number | - | - |
| 会員名 | string | - | - |
| 計算月 | string | ○ | ○ |
| 料金区分 | enum | - | - |
| 計算金額 | number | ○ | ○ |

---

## API利用

**計算**: POST /api/v1/care-costs/calculate

**リクエスト**:
```json
{
  "month": "string (YYYY-MM)",
  "pricingType": "enum"
}
```

---

## 操作フロー

```
START → [計算フォーム表示]
  → [計算月、料金区分を選択]
  → [計算ボタン押下] → POST /calculate
    ├→ [成功] → 計算結果表示
    └→ [失敗] → エラーメッセージ
  → [結果を確認・修正]
  → [承認ボタン押下]
  → END
```

---
