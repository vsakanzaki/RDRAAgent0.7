# UI仕様書: 売上管理画面

**画面日本語名**: 売上管理画面  
**画面英語名**: RevenueManagementScreen  
**所属BUC**: CollectionManagementFlow  

**目的**: 集計された売上情報を確認。事業所別の売上実績を把握・分析し、経営レポートを作成。

---

## 表示/入力項目

| 項目名 | 型 | 必須 | 編集可否 |
|-------|---|-----|--------|
| 集計月 | string | - | - |
| 事業所名 | string | - | - |
| 売上金額 | number | - | - |
| 回収金額 | number | - | - |
| 回収率 | number | - | - |

---

## API利用

**集計**: GET /api/v1/revenue?month={month}&businessLocation={location}

**詳細**: GET /api/v1/revenue/detail/:month

---

## 操作フロー

```
START → [集計月選択]
  → [売上情報取得] → GET /api/v1/revenue
  → [売上実績一覧表示]
  → [詳細分析]
  → [経営レポート作成]
  → END
```

---

## 機能要求

| 要求ID | 要求内容 | 優先度 |
|-------|--------|-------|
| FR-001 | 売上情報を確認できること | 必須 |
| FR-002 | 事業所別の売上実績を把握できること | 必須 |
| FR-003 | 経営レポートを作成できること | 推奨 |

---

## 監査/権限制御
