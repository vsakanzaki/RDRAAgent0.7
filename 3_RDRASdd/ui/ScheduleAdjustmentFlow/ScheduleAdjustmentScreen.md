# UI仕様書: スケジュール調整画面

**画面日本語名**: スケジュール調整画面  
**画面英語名**: ScheduleAdjustmentScreen  
**所属BUC**: ScheduleAdjustmentFlow  

**目的**: 確定したスケジュールを確認し、変更要望に基づいて調整。調整内容を確認。

---

## 表示/入力項目

| 項目名 | 型 | 必須 | 編集可否 |
|-------|---|-----|--------|
| スケジュールID | number | - | - |
| 会員名 | string | - | - |
| 現計画日時 | datetime | - | - |
| 調整案 | string | ○ | ○ |
| 調整結果 | enum | ○ | ○ |

---

## API利用

**調整**: PATCH /api/v1/schedules/:scheduleId/adjust

**リクエスト**:
```json
{
  "adjustmentProposal": "string",
  "adjustmentResult": "enum"
}
```

---

## 操作フロー

```
START → [スケジュール表示]
  → [調整案、結果を入力]
  → [調整ボタン押下] → PATCH /:id/adjust
    ├→ [成功] → 成功メッセージ
    └→ [失敗] → エラーメッセージ
  → END
```

---

## 機能要求

| 要求ID | 要求内容 | 優先度 |
|-------|--------|-------|
| FR-001 | スケジュールを調整できること | 必須 |
| FR-002 | 調整内容を確認できること | 必須 |
