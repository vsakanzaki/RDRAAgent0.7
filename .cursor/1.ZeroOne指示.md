# RDRAZeroOne

## 実施内容
- `RDRA_Knowledge/0_ZeroOne/phase1~phase5/`の配下の各mdファイルについて以下の処理を繰り返す
- git worktreeを作成しその中でAIを使ってファイル生成し作成したgit worktreeを削除する

### 詳細
- ファイル生成が完了したら、要約せずに停止してください
