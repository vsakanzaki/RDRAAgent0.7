# RDRAZeroOne フェーズごとの要件定義

## 目的
RDRAモデルをPhase1からPhase4までPhase毎に段階的に生成します
Phase毎の成果物がない、そのPhaseが実行される
Phase5は生成された成果物を他のフォルダーにコピーする

## 実施内容
- git worktreeを作成しその中でAIを使ってファイル生成し作成したgit worktreeを削除する
- `.cursor/1.ZeroOne指示.md`を読み取ってください
- Phase毎に実行する

## 留意事項
- 出力ファイルが既に存在する場合は生成をスキップし、存在しない場合は作成してください
- ファイル生成が完了したら、要約せずに停止してください

## エラー処理仕様
### ファイル作成失敗時
- 原因を特定し、具体的な修正方法を提示
- 前フェーズの出力不備の場合は該当フェーズの再実行を促す

### フォーマット不正時
- 不正行を特定し、正しいフォーマット例を提示
- 部分修正が可能な場合は修正案を提示

# 実行単位
- phaseが指示された時はそのPhaseだけを実行する　
- ファイルが指示された時はそのファイルだけを実行する
- phaseが指示されないときはphase1からphase5のうち定義されていないphaseを一つ実行する
- 実行後出力したファイルの内容を確認し、必要があれば修正するようにメッセージを表示する
