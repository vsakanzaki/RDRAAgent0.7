# verify-file

渡されたファイルについて、記述の矛盾や不正確性、参照の不正を検証してください

## 検証項目

1. **記述の矛盾**: 同一ファイル内または関連ファイル間で、内容が相互に矛盾していないか
2. **不正確性**: 事実として誤っている記述、定義と実態が一致しない記述がないか
3. **参照の不正**: 存在しないファイル・セクション・IDへの参照、リンク切れ、未定義の用語や変数への参照がないか

## 出力形式

問題が見つかった場合は以下の形式で報告してください：

### 種別
- 矛盾 / 不正確 / 参照不正
#### 箇所
- 行番号
#### 内容
- 題の説明
#### 修正案
- 推奨される修正内容

### 問題がない場合
-　「検証完了：問題は見つかりませんでした」と報告してください。
