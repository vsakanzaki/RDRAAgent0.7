# phase1：ビジネスパラメータ
- 「初期要望.txt」からビジネスパラメータを洗い出し「ph1ビジネスパラメータ.tsv」を出力する

# 背景理解
- 初期要望は実現したいビジネスとそれを実現するシステムについて記述している
- ビジネス上の様々な状況に対応できるように区分や分類を用いる

## ビジネスパラメータの洗出し
- 初期要望からビジネス上必要になる区分、分類、種別などのパラメータを洗い出す
- ビジネスパラメータが記述していない場合は記述されていることから、必要となるビジネスパラメータを類推する
- システムの仕組上の区分、分類、種別は含めない
-「ビジネスパラメータ」出力フォーマットで出力する

## 実行ルール
- 初期要望からバリエーションが導出できないときはヘッダーだけをファイルに出力する
- 出力内容について判断に迷った場合は可能性の一番高いものを選択する

## 出力ルール
- 出力には必ずヘッダー（カラム名）行を出力する
- 空フィールドにするときは、タブ区切りを省略しない
- プログラムは絶対に生成しない。

### 「ビジネスパラメータ」出力フォーマット
出力先：0_RDRAZeroOne/phase1/ph1ビジネスパラメータ.tsv
出力形式：TSV（タブ区切り）
カラム構成：
1列目: ビジネスパラメータ：ビジネスを行う上で必要な分類、区分、種別などを表す
2列目: 説明：ビジネスパラメータが必要な背景を説明する
- ヘッダー行を含め、各フィールドはタブ文字で区切ってください。
