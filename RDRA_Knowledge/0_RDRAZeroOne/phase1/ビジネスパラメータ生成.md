# phase1：ビジネスパラメータ
- 「初期要望.txt」から情報を洗い出す

# 背景理解
- 初期要望は実現したいビジネスとそれを実現するシステムについて記述している
- ビジネス上の様々な状況に対応できるように区分や分類を用いる

## ビジネスパラメータの洗出し
- 初期要望からビジネス上必要になる区分、分類、種別を洗い出してください
- ビジネスパラメータが記述していない場合は記述されていることから、必要となるビジネスパラメータを類推する
- システムの仕組上の区分、分類、種別は含めない
-「ビジネスパラメータ」フォーマットで出力する

## 出力ルール
- 出力には必ずヘッダー（カラム名）行を出力する
- 前後に「```tsv」「```」を付ける
- 「ビジネスパラメータ」以外の説明を出力しない

### 「ビジネスパラメータ」のフォーマット
- ビジネスを進めるうえで必要な分類、区分、種別などのパラメータを表す
- ２カラムのタブ区切りで出力する
```tsv
「ビジネスパラメータ、説明」
- ビジネスパラメータ：ビジネスを行う上で必要な分類、区分、種別などを表す
- 説明：ビジネスパラメータが必要な背景を説明する
```
