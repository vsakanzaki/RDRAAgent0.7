# ビジネスルール定義

## ビジネスルール一覧

### 1. サービススタッフスキル適合条件
**目的**: 介護会員の状況（要介護度）や要求されるサービス種別に対して、スタッフが適切なスキル（資格）を持っているかを判定する。

| サービス種別 | 介護会員の状況（要介護度） | 必須スキル（資格） |
| :--- | :--- | :--- |
| 身体介護 | 要介護3以上 | 介護福祉士、実務者研修修了 |
| 身体介護 | 要介護1〜2 | ヘルパー1級、初任者研修修了 |
| 生活援助 | 全区分 | ヘルパー2級以上 |
| 通院介助 | 全区分 | 初任者研修修了以上 |

### 2. 介護費用計算（サービス単価）
**目的**: サービス種別と時間帯に基づいた基本単価を定義する。

| サービス種別 | 時間帯 | 単位時間(30分)単価 |
| :--- | :--- | :--- |
| 身体介護 | 日中(8:00-18:00) | 4,000円 |
| 身体介護 | 早朝夜間 | 5,000円 |
| 生活援助 | 全時間帯 | 2,500円 |
| 夜間対応型 | 深夜 | 6,000円 |

### 3. スタッフ働き方制限
**目的**: 働き方分類に基づいた勤務時間の制限を行う。

| 働き方分類 | 1日の最大勤務時間 | 週の最大勤務日数 |
| :--- | :--- | :--- |
| 常勤フルタイム | 8時間 | 5日 |
| 常勤パートタイム | 6時間 | 4日 |
| 非常勤 | 4時間 | 3日 |
| 登録ヘルパー | 制限なし（シフト制） | 制限なし |

### 4. 介護施設分類によるサービス提供制限
**目的**: 介護会員が入居・利用している施設タイプによって提供可能なサービスを制限する。

| 介護施設分類 | 身体介護 | 生活援助 | 通院介助 |
| :--- | :--- | :--- | :--- |
| 在宅 | 可 | 可 | 可 |
| デイサービス | 不可 | 不可 | 可 |
| グループホーム | 可（一部） | 不可 | 可 |
| 特別養護老人ホーム | 不可 | 不可 | 不可（施設内対応） |

### 5. 請求区分による負担率
**目的**: 請求先区分（介護保険・自費）による利用者負担額を算出する。

| 請求先区分 | 利用者負担率 | 備考 |
| :--- | :--- | :--- |
| 介護保険 | 10%〜30% | 所得に応じて変動 |
| 自費 | 100% | 全額自己負担 |
